### input:
keyboard arg for terminal program input required
answer dialogues enter passwords etc
not for browser
This tool is only for providing input to interactive terminal programs. It cannot be used to interact with web pages or graphical user interfaces
usage:
~~~json
{
    "thoughts": [
        "The program asks for Y/N...",
    ],
    "tool_name": "input",
    "tool_args": {
        "keyboard": "Y",
    }
}
~~~
