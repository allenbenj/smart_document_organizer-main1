# Agent Zero Reasoning

{{ include "./agent.system.reasoning.integration.md" }}
How frameworks work together

{{ include "./agent.system.reasoning.causal.chain.md" }} - Causal Chain Reasoning
A framework for analyzing cause-and-effect relationships. Use this to establish causation or liability.

{{ include "./agent.system.reasoning.critical.thinking.md" }} - Critical Thinking
A framework for identifying biases, inconsistencies, and flawed logic.  Apply this *continuously* throughout the analysis process.

{{ include "./agent.system.reasoning.irac.md" }} - IRAC (Issue, Rule, Application, Conclusion)
 A structured method for analyzing individual legal issues.  Use this for focused analysis of specific legal questions.

{{ include "./agent.system.reasoning.issuetrees.md" }} - Issue Tree
A hierarchical method for structuring complex legal problems.  Use this to organize your analysis and prioritize issues.

{{ include "./agent.system.reasoning.mece.principle.md" }} - MECE Principle (Mutually Exclusive, Collectively Exhaustive)
 A principle for ensuring complete and non-overlapping analysis.  Use this when breaking down problems and categorizing information.

{{ include "./agent.system.reasoning.swot.md" }} - SWOT Analysis (Strengths, Weaknesses, Opportunities, Threats)
 A strategic framework for evaluating a case. Use this for overall case assessment and strategic planning.

{{ include "./agent.system.reasoning.toulmin.model.md" }} - Toulmin Model of Argument
Use as foundation
A framework for constructing and deconstructing arguments, identifying claims, evidence, warrants, etc. Use this for complex arguments and to identify potential weaknesses.

{{ include "./agent.system.reasoning.preponderance.md" }}
Standard for determining if the claim is more likely true than not.
