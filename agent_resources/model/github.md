## GitHub Models

A set of models are available on GitHub, with the same name/capabilities/metrics as Microsoft Foundry (AI Toolkit) models, except for 'Cost' and 'Context Size'.
  - GitHub models are fully supported in AI Toolkit as well.
  - GitHub models are free to start, or enable billing later to avoid rate limits.
  - GitHub models have less context size compared to Foundry deployments.
  - For development, GitHub models have a single endpoint (\`https://models.github.ai/inference/\`) with model ID in parameter, and use GitHub [PAT](https://docs.github.com/en/authentication/keeping-your-account-and-data-secure/managing-your-personal-access-tokens) for authentication.
